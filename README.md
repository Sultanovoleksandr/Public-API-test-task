# Public-API-test-task

Цей проєкт містить колекцію автоматизованих тестів для публічного API, реалізованих за допомогою **Postman**. 

## 📋 Опис проєкту
Проєкт створений для демонстрації навичок тестування API, включаючи:
* Перевірку статус-кодів та типів даних.
* Роботу зі змінними оточення та колекцій.
* Динамічну генерацію тестових даних.
* Перевірку структури JSON-відповідей.

## 🛠 Технології
* **Postman** — основний інструмент тестування.
* **JavaScript** — для написання тестів та скриптів (Pre-request/Tests).

## 🚀 Як запустити тести

Ви можете запустити ці тести локально, виконавши наступні кроки:

### 1. Імпорт у Postman
1. Скачайте файл [Test task.postman_collection.json](https://github.com) з цього репозиторію.
2. Відкрийте Postman.
3. Натисніть кнопку **Import** у верхньому лівому куті.
4. Перетягніть скачаний JSON-файл у вікно імпорту.

### 2. Запуск тестів
Ви можете запускати запити двома способами:
* **Окремо**: Виберіть будь-який запит і натисніть кнопку **Send**. Результати тестів будуть доступні у вкладці *Test Results*.
* **Масово (через Runner)**: 
  1. Натисніть правою кнопкою миші на назву колекції.
  2. Оберіть **Run collection**.
  3. Виберіть потрібну кількість ітерацій (наприклад, 3) та натисніть **Run**.
